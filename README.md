# API de Estudos

Esta é uma API desenvolvida como parte dos meus estudos durante a pós-graduação em Desenvolvimento de Software com Golang.

## Tecnologias Utilizadas

- Golang

## Como Executar

1. Clone o repositório:+
2. Navegue até o diretório do projeto:
    ```sh
    cd api-pos
    ```
3. Execute a aplicação:
    ```sh
    go run main.go
    ```
